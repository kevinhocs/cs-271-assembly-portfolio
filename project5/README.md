\## Project 5 - String Primitives and Macros



x86 assembly program that reads comma-delimited temperature data from a file, parses values into signed integers using custom macros and procedures, and outputs the data in reverse order.



\### Concepts

\- File I/O and buffered input processing

\- Custom macros for string input and output abstraction

\- Manual string parsing and integer conversion

\- Stack-based procedures with explicit parameter passing

\- Array traversal and reverse-order output formatting
